Rust implementation of Amazon's Ion data format

Ion is a data format very similar to JSON, with enhancements such as additional types (timestamps, s-expressions, etc), typed null values (an unset integer would be null.int), and ability to add annotations to all values.

This implementation is very alpha and subject to massive changes. Help and suggestions in its development would be appreciated.

Learn more by checking out the specification and the java implementation.
